# Iziwork — Cas pratique Senior PM

Infographies HTML autonomes produites en préparation du cas pratique Senior PM (scope Staffing) chez Iziwork.

- **`funnel-iziwork.html`** — Premier schéma du funnel : 5 étapes officielles du cas pratique + 4 étapes du cycle de vie omises (suivi de mission, renouvellement, fin de mission, resollicitation), avec statut IA par étape.
- **`funnel-baseline-manuel.html`** — Frise chronologique du process 100% manuel (avant toute brique IA) : charge manuelle par étape avec détail acteurs/outils/données/frictions/risques.
- **`funnel-iziwork-brand.html`** — Version finale, aux couleurs Iziwork : toggle Avant / Maintenant / Après (charge manuelle, délai cumulé de staffing, qualité perçue client, deltas vs baseline), backlog d'opportunités IA priorisé par score RICE, et alerte de contrôle humain obligatoire sur l'étape Qualification.

Chaque fichier est un artifact HTML autonome (CSS/JS inline, aucune dépendance externe) — ouvrable directement dans un navigateur.
